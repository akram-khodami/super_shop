# Supermarket E-Commerce

Laravel based multilingual e-commerce platform.

## Features

### Customer

- Product listing
- Product details
- Shopping cart
- Checkout
- Address management
- Order management
- Wallet
- Online payment
- Order tracking

### Admin

- Product management
- Inventory management
- Order management
- Order status updates
